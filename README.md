# AutoGen Multi-Agent Platform

## 项目愿景
打造下一代多智能体协同平台，模拟真实组织中的跨角色协作，服务亿级营收场景。

## 目录结构
- apps/frontend/  —— 前端（Next.js + Tailwind）
- apps/backend/   —— 后端（FastAPI + AutoGen）
- packages/agents/—— Agent 定义与角色逻辑
- packages/tools/ —— 公共工具模块
- docs/           —— 结构化文档
